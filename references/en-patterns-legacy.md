# English / legacy patterns

Use this reference only when the text is English or the user explicitly asks for English.

## High-signal AI markers
- inflated significance: `pivotal`, `testament`, `underscores`, `evolving landscape`;
- vague attribution: `experts say`, `industry reports`, `observers note`;
- promotional language: `groundbreaking`, `seamless`, `vibrant`, `rich`, `must-visit`;
- participle padding: `highlighting`, `ensuring`, `reflecting`, `showcasing`;
- copula avoidance: `serves as`, `stands as`, `boasts`, `features`;
- rule of three and synonym cycling;
- `not only... but also`, `not just... it is`;
- em dash overuse;
- bolded inline headers;
- chatbot artifacts: `Of course`, `I hope this helps`, `let me know`;
- generic upbeat conclusions.

## Correction
- prefer concrete verbs and details;
- remove fake importance;
- cite exact sources if present;
- preserve tone and technical meaning;
- vary rhythm without forcing personality.
