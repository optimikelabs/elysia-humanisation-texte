# Typographie française

## Quand l'appliquer
Appliquer si le texte est un contenu final en français : page, email, post, note, document.

## Prudence
Ne pas modifier :
- code;
- commandes;
- URLs;
- slugs;
- clés JSON/YAML;
- extraits Markdown sensibles;
- texte volontairement ASCII.

## Points utiles
- Accents sur majuscules quand c'est du texte final : `État`, `À propos`, `Ça`.
- Espaces avant `:`, `;`, `?`, `!` si le support les accepte.
- Guillemets français si le niveau éditorial le justifie.
- Pas de virgule avant `et` dans une énumération simple.
- Cohérence des apostrophes dans un même texte.
- Pas de tiret cadratin `—` dans un texte final français courant, sauf dialogue ou citation qui l'exige. Remplacer par un point, une virgule, des parenthèses ou un deux-points selon le sens.
- Après un deux-points, garder la minuscule sauf nom propre, sigle, citation directe ou titre volontaire.
- Dans les titres français, éviter le title case : seul le premier mot et les noms propres prennent une majuscule.

## Règle
La typographie sert la finition; elle ne doit jamais devenir plus importante que le sens, le registre ou la compatibilité technique.
